# Kasumi界面工具和Akatsuki界面代码生成工具

Kasumi界面工具是用来通过XML文件编辑多分辨率相对布局的工具。
Akatsuki界面代码生成工具是用来生成对应的二进制数据和控件绑定代码。
Akatsuki现在支持cocos2d-x的控件绑定代码生成。
